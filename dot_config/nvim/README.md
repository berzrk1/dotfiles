# Neovim

- Powered by [LazyVim](https://github.com/LazyVim/LazyVim)

## Plugins

- **editor**
  - `devdocs`: DevDocs browser
- **lang**
  - `bash`: Bash support with my configurations
  - `c`: C support with my configurations
  - `cpp`: Cpp support with my configurations
  - `docker`: Docker support with my configurations
  - `fish`: Fish support with my configurations
  - `hypr`: Hyprland support with my configurations
  - `json`: Json support with my configurations
  - `markdown`: Markdown support with my configurations
  - `python`: Python support with my configurations
  - `toml`: Toml support with my configurations
  - *web*: Web development related languages
    - `html-css`: HTML/CSS support with my configurations
    - `php`: PHP support with my configurations
    - `python`: Python Django support with my configurations
    - `tailwind`: Tailwind support with my configurations
    - `typescript`: Typescript support with my configurations
- **LSP**
  - `otter`: LSP features inside embedded code blocks (e.g. code in markdown)
- **UI**
  - `colorful-winsep`: Colored window border separator
  - `colorizer`: Inline CSS color highlighting with a toggle keybinding
  - `diagflow`: LSP diagnostics rendered at the top-right corner instead of inline
  - `guess-indent`: Auto-detects indentation style per file
  - `log-highlight`: Syntax highlighting for log files
  - `noice-extended`: Add a border to LDP doc
  - `colorschemes/neofusion`: colorscheme
  - `colorschemes/eidolon`: colorscheme
- **util**
  - `cloak`: Hide secrets in `.env` files (there is a toggle keybinding)
  - `hardtime`: Notification about bad habits in Vim (e.g. repeated keys)
  - `image.lua`: Render images inline
  - `img-clip`: Pastes clipboard images into documents
  - `leetcode`: LeetCode integration
  - `rest-extended`: REST Client inside Nvim
  - `tmux`: Tmux and Nvim integration with window navigation and resizing
  - `wakatime`: Track coding activity using Wakatime
  - `yazi`: Use Yazi as the file explorer, replacing the default Lazyvim Snacks
